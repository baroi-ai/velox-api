# ⚡ Velox API
**High-Performance, Offline-First API Testing Suite.**

Velox API is a lightweight, blazing-fast alternative to bulky API clients. Designed for developers who value performance and simplicity, it allows you to test endpoints, manage collections, and import Swagger/OpenAPI documentation—all with zero lag and total privacy.

## ✨ Key Features

* 🚀 **Native Performance:** Built with Next.js for nearly instant interaction handling and state updates.
* 🌐 **Offline-First:** No accounts needed. All your request history and collections are stored securely in your browser's local storage.
* 📑 **Smart Import:** Seamlessly transition from other tools by importing **Swagger**, **OpenAPI**, or **Postman** collections.
* 💎 **JSON Beautifier:** Built-in syntax highlighter and formatter for handling complex JSON payloads.
* 📱 **Fully Responsive:** Use the power of a professional API tester on your Desktop, Tablet, or Mobile phone.
* 🛡️ **Privacy Oriented:** No telemetry. No tracking. Your API keys and data never leave your machine.

## 🚀 Getting Started (Run Locally)

Run your own instance of Velox API or contribute to the project by following these steps:

### Prerequisites
* **Node.js 18.x** or higher
* **npm**, **pnpm**, or **bun**

### Installation

1.  **Clone the repository**
    ```bash
    git clone [https://github.com/baroi-ai/velox-api.git](https://github.com/baroi-ai/velox-api.git)
    cd velox-api
    ```

2.  **Install dependencies**
    ```bash
    npm install
    ```

3.  **Run the development server**
    ```bash
    npm run dev
    ```

4.  **Open the App**
    Navigate to [http://localhost:3000](http://localhost:3000) in your browser.

## 🛠️ Tech Stack

- **Framework:** [Next.js 15+](https://nextjs.org/) (App Router)

- **State Management:** [Zustand](https://github.com/pmndrs/zustand)
- **Animations:** [Framer Motion](https://www.framer.com/motion/)
